About

What is Eurydice?

Eurydice is a hybrid semantic search engine for Amazon Fashion products. It combines dense vector retrieval with structured filter extraction to match natural-language queries — like "affordable rain boots for women" — to the most relevant clothing items.

The pipeline uses Gemini Flash to generate synthetic training data, fine-tunes a sentence transformer for embedding-based similarity, and trains a Flan-T5 model to extract structured filters (price, rating, subcategory) from free-text queries.
